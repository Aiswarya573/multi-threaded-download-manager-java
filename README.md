# Multi-Threaded Download Manager (Java)

##  Overview
This project simulates a browser-like download manager using Java multithreading.

##  Features
- Multiple file downloads using threads
- Thread lifecycle demonstration (NEW → RUNNABLE → TERMINATED)
- Thread synchronization using join()
- Thread status tracking using isAlive()
- Thread priority handling

##  Technologies Used
- Java
- Multithreading

##  How to Run
1. Compile:
   javac DownloadManager.java
2. Run:
   java DownloadManager

##  Output
Displays parallel file download simulation with progress updates.
